# grpc-microservices-demo

grpc-java demo.

[gRPC repository](https://github.com/grpc/grpc)

[gRPC Java repository](https://github.com/grpc/grpc-java)

### Running services

#### Unary

```
./gradlew runServer
```

```
./gradlew runClient
```

#### Streaming

```
./gradlew streamingServer
```

```
./gradlew streamingClient
```
